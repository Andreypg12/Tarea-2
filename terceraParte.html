Explicación algunos de los elementos:

  • flex-flow: El flex-flow lo que hace es combinar el flex-direction con el flex-wrap, podemos poner flex-flow: row wrap; y de esa forma hacer la combinación que queramos.
  • order: En el codigo de esta página se hizo caja1, caja2, caja3 en ese orden, pero se invirtio usando el order para que a nivel visual se vea primero la caja3, luego la caja2 y por último la caja1. El order es un número, y el que tenga el número más bajo va a aparecer primero.
  • flex-shrink: La propiedad flex-shrink controla cuánto puede encogerse un ítem dentro de un contenedor flex cuando el espacio disponible es limitado, si tenemos un elemento con este valor en 0 no se encogerá, si está en 1 se encogerá normal, y entre más alto sea el número cada vez se concogerá más rápido.
  • align-self: La propiedad aling-self permite sobrescribir el valor de align-items para un ítem específico.

¿Que significa el "Flex Responsive"?

El flex responsive no es una propiedad, es un enfoque o técnica que aprovecha Flexbox para lograr diseños adaptativos o responsivos,
lo que logra que las páginas se ajusten automáticamente al tamaño de la pantalla (como en celulares, tablets o computadoras).
